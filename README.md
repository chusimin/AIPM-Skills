# Skills

我的 Claude Cowork Skills 合集 —— 一套为 AI 产品经理求职与个人成长打造的自定义技能包。

## 这是什么？

这些是 [Claude Cowork](https://claude.ai) 的自定义 Skill 文件。每个 Skill 是一份结构化的 Prompt，让 Claude 在特定场景下变成专业的助手，而不是泛泛而谈的通用 AI。

把它们装进 Cowork 后，只需要用自然语言触发（比如"帮我看看面试"、"写个PRD"），Claude 就会自动加载对应的专业流程。

## 包含哪些 Skill？

| Skill | 干什么用 |
|-------|---------|
| **ai-pm-interview-diagnosis** | 面试诊断与备战。上传面试记录，逐题分析面试官意图、找出回答中的 Gap、给出可练习的改进方案。也支持基于 JD 的模拟备战。 |
| **ai-pm-resume-writer** | AI PM 简历撰写与优化。从零写简历、诊断现有简历、针对目标 JD 做适配调整，覆盖 15-40k 全薪资段。 |
| **ai-agent-prd-writer** | AI 产品 PRD 撰写。支持工具型、Agent 型、对话型、平台型等各类 AI 产品的需求文档，从模糊想法到完整 PRD。 |
| **interactive-learning** | 一对一自适应学习引擎。基于 Bloom 2 Sigma 理论，用苏格拉底式追问帮你真正理解一个知识领域，而不是被动灌输。 |
| **obsidian-knowledge-saver** | 对话沉淀到 Obsidian。把和 Claude 的对话精华自动提炼为结构化笔记，带标签和双向链接，写入本地知识库。 |

## 怎么用？

1. 在 Claude Desktop 的 Cowork 模式中，将 Skill 文件（`SKILL.md`）添加到你的 Skills 目录
2. 正常和 Claude 对话，相关场景会自动触发对应 Skill
3. 也可以直接说触发词，比如"面试诊断"、"帮我写简历"、"我想学 XX"

## 关于

这些 Skill 是我在 AI 产品经理转型过程中逐步打磨出来的，解决的都是真实场景中的痛点。欢迎参考和改造。
