# AIPM+ Skills 工具箱

一套为 AI 产品经理打造的 Claude Skills，覆盖从知识学习、产品拆解、简历撰写、面试诊断到知识沉淀的完整链路。

这几个 Skill 组合在一起，足够支撑你从「准备转型」到「拿到 offer 开始上手工作」的整个过程。

## 为什么做这套 Skills？

市面上大多数 AI 求职工具要么是通用模板（"帮你润色简历"），要么是信息聚合（"AI 面试题大全"）。它们的问题在于：**没有针对 AI 产品经理这个岗位的专业判断力。**

比如你让通用 AI 帮你改简历，它会帮你把句子写得更漂亮，但不会告诉你"这段经历应该突出数据闭环而不是功能迭代，因为目标 JD 要的是有 AI 评测体系经验的人"。

**这套 Skills 的不同之处：**

- **内置 AI PM 领域知识**：每个 Skill 都内嵌了 AI 产品经理岗位的能力模型、薪资分层标准（15-40k）、面试官考察维度等专业框架
- **流程驱动而非问答驱动**：触发后 Claude 会按照完整的专业流程主动推进
- **实战验证**：每个 Skill 都经历了多轮真实求职场景的测试和迭代

### 关于 obsidian-knowledge-saver（v2.0 重构版）

这个 Skill 刚做了一次大重构，核心变化是：

**存的是知识点，不是对话记录。** 你聊了一小时可能聊了5个话题，旧版是把整个对话打包成一个大文件塞进去。新版是拆成5条独立的小笔记，每条只讲一件事。这样以后你想找某个知识点，直接找那一条就行，不用翻一大篇对话记录去定位。

**同一个知识点只有一条笔记。** 你今天聊了"Kano模型"，存了一条。下周又聊到"Kano模型"有了新理解，不是再存一条新的，而是更新那条已有的。关于 Kano 模型你永远只需要看一个地方，而且那个地方永远是最新最全的。

**整理的活儿 LLM 干，你只管说"存"。** 该放哪个文件夹、该打什么标签、该和哪些笔记建立链接、索引要不要更新——这些你都不用操心，LLM 全包了。你唯一要做的就是看一眼确认"对，就这么存"。

新版还支持四种操作模式：
- **沉淀（Save）**：聊完说"帮我存一下"，自动拆成原子笔记
- **消化（Ingest）**：丢一篇文章说"帮我消化"，自动提炼知识卡片
- **健康检查（Lint）**：说"检查一下知识库"，自动找出孤立笔记、断裂链接、重复内容
- **初始化（Init）**：首次使用自动引导你设计目录结构和命名规范

理念来自 Zettelkasten 原子笔记方法和 [Karpathy 的 LLM Wiki 模式](https://gist.github.com/karpathy/442a6bf555914893e9891c11519de94f)。

## 快速开始

### 第一步：安装 Claude Desktop

前往 [claude.ai/download](https://claude.ai/download) 下载 Claude Desktop 客户端。

### 第二步：开启 Cowork 模式

打开 Claude Desktop → 左下角切换到 **Cowork** 模式。首次使用会引导你选择一个工作文件夹。

### 第三步：安装 Skills

1. 点击本仓库页面的绿色 **Code** 按钮 → **Download ZIP**，解压到任意位置
2. 在 Cowork 模式下，点击左侧边栏的 **Skills** → **Add Skill**
3. 选择你要安装的 Skill 文件夹（比如 `ai-pm-resume-writer`），每个文件夹是一个独立的 Skill
4. 安装完成后，Skill 会自动加载，无需重启

### 第四步：直接用自然语言触发

不需要记任何命令。直接跟 Claude 说你想做的事，对应的 Skill 会自动触发：

- "帮我看看这个面试记录" → 自动触发**面试诊断**
- "我想投这个岗位，简历怎么改" → 自动触发**简历优化**
- "帮我挖掘AI写作工具的用户痛点" → 自动触发**产品灵感挖掘**
- "帮我拆解一下 Kimi" → 自动触发**产品拆解**
- "帮我写个 PRD" → 自动触发 **PRD 撰写**
- "帮我48小时搞懂多模态" → 自动触发**加速学习**
- "我想搞懂 RAG" → 自动触发**学习引擎**
- "沉淀到知识库" → 自动触发**知识沉淀**
- "帮我消化这篇文章" → 自动触发**知识消化**
- "lint一下知识库" → 自动触发**知识库健康检查**

## 推荐使用路径

```
1. 速成领域 → 48h-accelerated-learning（48小时建立目标领域的认知全景图）
2. 补知识 → interactive-learning（系统深入搞懂 RAG、Agent、评测体系等核心概念）
3. 挖需求 → product-insight-miner（从小红书/X/Reddit挖掘用户痛点和产品机会）
4. 看产品 → ai-product-teardown（拆解目标公司和竞品的 AI 产品）
5. 攒作品 → ai-agent-prd-writer（写出能拿得出手的 AI PRD）
6. 写简历 → ai-pm-resume-writer（针对目标 JD 打磨简历）
7. 打面试 → ai-pm-interview-diagnosis（面试后复盘，面试前备战）
8. 全程沉淀 → obsidian-knowledge-saver（把学到的东西拆成知识卡片存下来）
```

入职之后，**product-insight-miner** 帮你做市场调研和需求发现，**48h-accelerated-learning** 帮你快速搞懂新业务领域，**ai-product-teardown** 和 **interactive-learning** 继续帮你理解新产品和新技术，**obsidian-knowledge-saver** 帮你持续积累和管理项目知识。
